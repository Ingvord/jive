Find a property

Go in the Edit menu and select "Find a property". This will open the search window. Type the text to find and click "Start".

"Ignore case" allows to perform a case idependant search.
"Search in property value" allows to search also in property value but this may slow down the process.
"Search in attribute property" allows to search in attribute properties. But as the attribute list is returned by the server. You will wait for all timeouts on each device included in the search (if this device is not running)
"Use regular expresion" allows to use regular expression in the search.
