How to create a Tango Server

To Create a server , Go to the "Edit" menu and select "Create server". This will open the server creation window.

 

 

 

1) Enter the server name (Executable name) including its instance name. Ex: MyServer/1 where MyServer is the name of your process and 1 its instance.
2) Enter the class name
3) Enter all devices for this class
4) Click on Register server

 

 

Managing servers

Select the server (in the server tree) you want to edit and right click on it to get this menu.
Rename: Rename a server instance.
Delete: Remove a server instance.
Add class: This will prompt the server creation window with the server name field locked. Enter the class and device(s) names and click on "Register Server".
Test admin server: Open a device panel on the admin device.
Classes Wizard: Launch the wizard on classes of this server.
Unexport device: Unexport all devices belonging to this server.

Managing classes

Select the class (in the server tree) you want to edit and right click on it to get this menu.
Rename: Rename a class.
Delete: Delete a class.
Add device: This will prompt the server creation window. with the server name and class name locked. Enter the device name(s) and click on "Register server".
Devices wizard: Launch the wizard on devices belonging to this class.

Managing devices

Select the device (in the server tree) you want to edit and right click on it to get this menu.
Copy: Copy the whole device property set to the clipbaord.
Paste: Copy all properties within the clipboard to the device.
Rename: Rename a device.
Delete: Delete a device.
Monitor device: Launch AtkPanel on this device.
Test device: Launch the Device panel on this device.
Define device alias: Define an alis for this device.
Go to device node: Select the device within the device tree.
Restart device: Restart a device (by calling DevRestart on the admin device)
Device wizard: Launch the wizard on this device.
