Creating/Editing/Deleting a Property

Go in the device tree (or server tree) and select the "Properties" node of a device. This will display the property panel in the right part of the application.

 

Then, you can change a value of a property by typing your data in the "Value" field and click on "Apply".Each line of text (for a same property) is interpreted as a string array. You can create a new property by clicking on the "New property" button and you can delete a property by clicking on the "Delete" button.

 

 

 

Renaming a Property

Rigth click on the property and select "Rename", This will allow you to give a new name to your property.
