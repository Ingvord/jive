Managing events

Go in the device tree (or server tree) and select the "Event" node of a device. This will display the event configuration panel in the right part of the application.

 

 

Then, you can enter the absolute or relative change event property for an attribute.
Relative: - a property of maximum 2 values (separated by a coma). It specifies the positive and negative relative change of the attribute value w.r.t. the value of the previous change event which will trigger the event. If the attribute is a spectrum or an image then a change event is generated if any one of the attribute value's satisfies the above criterium. If only one property is specified then it is used for the positive and negative change.
Absolute: - a property of maximum 2 values (separated by a coma) .It specifies the positive and negative absolute change of the attribute value w.r.t the value of the previous change event which will trigger the event. If the attribute is a spectrum or an image then a change event is generated if any one of the attribute value's satisfies the above criterium. If only one property is specified then it is used for the positive and negative change. If no properties are specified then the relative change is used.
