Select multiple properties

You can select several properties by holding the CTRL key down. Then the property value text area will show you a value (if all properties have the same value) or "Values are not equals" (if al least one property differs from the others).
Now you can change the value of all these properties by typing the new value and click on the "Apply change" button. If one property differs, you can get a global vision of your selection by clicking on the "Show details" button.

Auto select multiple properties

Go in the edit menu and select "Mulitple selection". This will allow you to search and select some properties in the database. Note that you have to select first the root node of your search. In this example we select all properties named "voice_dev" in all SYS/MSTATUS/* devices.

You get the following window, enter the name of the property you want to select and click "Start". Further informations are available in the "Find a property" chapter.

 

This will auto select all properties named "voice_dev". You can now edit all of them by using the edit field.
