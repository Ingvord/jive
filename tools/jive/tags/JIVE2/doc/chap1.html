Create a Tango Server

To Create a server , you have to right click on the "SERVER" node in the tree and select create server. This will open the server creation window.

 

 

 

1) Enter the server name including its instance name. Ex: MyServer/1 where MyServer is the name of your process and 1 its instance.Then you can run your server by launching MyProcess 1.
2) Enter the class name
3) Enter all devices for this class
4) Click on Register server

 

 

Adding/Removing a device to/from an existing server

Select the class in the server you want to add a device, then right click on it and select "Add a device". This will ask you for a device name.
Select "Delete" if you want to remove the device. Note that when you remove a device all its properties are also removed.

 

Adding/Removing a class to/from a existing server

Select the server you want to add a class, then right click on it and select "Add class to server". This will show you the Create/Edit server window with the server name locked. Enter the class and devices name and click "Register Server".
